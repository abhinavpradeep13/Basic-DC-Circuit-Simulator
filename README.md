DC Circuit Simulator (OOP & Tkinter)
This project is a Python-based DC Circuit Simulator that allows users to design and analyze 
electrical networks consisting of resistors and independent voltage sources. It utilizes 
Object-Oriented Programming (OOP) principles to model circuit components and implements Modified 
Nodal Analysis (MNA) for its computational engine.The application features a graphical user interface (GUI) 
built with Tkinter, making it an intuitive tool for educational purposes or for validating small-scale DC circuit designs.
